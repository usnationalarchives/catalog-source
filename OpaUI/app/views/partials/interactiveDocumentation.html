National Archives Catalog API

This page provides interactive documentation to NARA's Online Catalog API, which allows you to programmatically access records, digital objects, and public contributions from the National Archives Catalog.
In addition, APIs for updating public contributions are provided. Please refer to the NARA's Online Catalog API Reference Guide for further details of API usage.

API Philosophy
NARA's Online Catalog API is parameter-based and supports the GET, POST, PUT, and DELETE methods:
  • GET for performing searches and retrieving metadata.
  • POST is used when creating new things (e.g. creating new public contributions)
  • PUT is used when updating something (e.g. saving public contributions)
  • DELETE is used when removing things (e.g. deleting public contributions)
