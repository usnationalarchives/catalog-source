POST Registration
Mode detail
To register in Catalog using NARA's Online Catalog API, you would need to use the following URL:
/users?userName={userName}&fullName={fullName}&displayFullName={true/false}&email={email}&password={password}&userType=standard&userRights=regular
Parameter Value Description
userName
Example: jsmith
Description Example: jsmith
fullName
Example: John Smith
Description Example: John Smith
displayFullName
If value is checked full name field will be required.
Description If value is checked full name field will be required.
email
Example: John Smith . If the email contains @nara.gov. This means is Nara Staff so fullname will be required
Description Example: John Smith
password
Password must contain minimum of 8 characters and can contain special characters or numbers
Description Password must contain minimum of 8 characters and can contain special characters or numbers
userType standard Constant value. It is not editable
Description Constant value. It is not editable
userRights regular Constant value. It is not editable
Description Constant value. It is not editable
