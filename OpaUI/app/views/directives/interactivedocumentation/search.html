GET Search
To perform a parameterized search in NARA's Online Catalog API, you would need to use the following endppoint:
?action=search&q={Your-Search-Term}
Enter a search term and click on submit to see the response from the API. You can choose one of the supported formats:
  • json (default response)
  • xml

This method require credentials. You will need to add them as part of the REST request.
Parameter Param Value Description
format
Format of the output response. Default value json.
Description
Format of the output response. Default value json.
rows
Number of results. Example = 20 , 50 , 100
Description
Number of results. Example = 20 , 50 , 100
offset
By default the results are returned from the first one until the end grouped by the number of rows requested. The beginning of the result index can be changed with this parameter. Default value is 0. Example = 20 , 50 , 100
