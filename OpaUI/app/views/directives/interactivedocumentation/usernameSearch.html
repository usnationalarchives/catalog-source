GET Username Search
To search for a Username in NARA's Online Catalog API, you would need to use the following endpoint:
/users?action=search&id={USERNAME}
Parameter Value Description
id
To search for a registered users username. User needs to be logged in as an Administrator to perform this action. Example: jsmith
