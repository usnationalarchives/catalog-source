POST Bulk Export
To perform a bulk export in Catalog using NARA's Online Catalog API, you would need to use the following endpoint:
?q={Your-Search-Term}&action=export

This method require credentials. You will need to add them as part of the REST request.
Parameter Param Value Description
rows
Number of results. Example = 20 , 50 , 100
Description
Number of results. Example = 20 , 50 , 100
offset
By default the results are returned from the first one until the end grouped by the number of rows requested. The beginning of the result index can be changed with this parameter. Default value is 0. Example = 20 , 50 , 100
